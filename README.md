# RankPilot Mobile v2.0

토스 디자인 시스템 + 카카오톡 공유 + 카드 패턴으로 풀 리빌드.

## 주요 변경 (v1.1.0 → v2.0)

### UI/UX 풀 리빌드
- 토스 디자인 시스템 적용 (색상 토큰 / 타이포 / 카드 패턴)
- 카드형 키워드 리스트 (한 키워드 = 한 카드)
- 분류 탭 (전체/조치/노출/24h)
- 5탭 하단 네비 (홈 / 그룹 / 월보장 / 내정보)
- 바텀시트 모달
- 키워드 행에 [캡처] [완료] [공유] 3-up 버튼

### 카톡 공유 (NEW)
- Web Share API 사용
- 사진 + 텍스트 한 번에 공유
- 건바이 보고 모달 (체크박스 다중 선택)
- 클립보드 폴백

### 매트릭스
- 모바일 최적화 7일 매트릭스
- 가로 스크롤 + sticky 키워드 컬럼

### 광고주 그룹 (admin)
- 광고주별 폴더 토글
- 광고주 그룹 카톡 보고
